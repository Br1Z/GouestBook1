Для разворачивания проекта нужен установленный Open Server
В PhpMyAdmin создать баззу даных c именем "reviews"
В программе используется ORM RedBeenPhp поэтому в файле \include\lib\db.php находятся настройки подключения 
к базе данных (mysql:host=localhost;dbname=reviews','root', '')

Поскольку в программе используется ORM RedBeenPhp таблицу создавать не нужно. При дабовлении данных 
она автоматически создаться в базе данных. 

Для добавления 50 искуственно созданных данных нажмите в правом верхнем углу на кнопку "Добавить 50 записей".
